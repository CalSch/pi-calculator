About

This website doesn't just calculate pi, it calculates pi many times with different accuracy, and plots the data on a graph.

The algorithm I use to calculate pi is a simple one I thought of (probably not very efficient):

  1. Make a half-circle with the function Math.sqrt(1-x**2)
  2. Make verts number of equally spaced vertical lines from -1 to 1 with this function: x=-1+(i*2)/(verts-1)
  3. Put all of the points where the lines intersect with the half-circle into the points array (I use "verts" and "points" interchangeably)
  4. Add up the distances between each point
  5. It's (close to) pi!
  6. The more vertices the more accurate

My JavaScript implementation is at pi.js
